## Key Concepts

### Self-Balancing BSTs
- AVL: Height-balanced with rotations
- Red-Black: Black-height balanced
- Treap: Randomized priorities
- Splay: Self-adjusting with splaying

### Range Query Structures
- Segment Tree: Range queries and updates
- Fenwick Tree: Prefix sums
- Sparse Table: Static RMQ

### Tree Decomposition
- LCA: Binary lifting, Euler tour + RMQ
- Heavy-Light Decomposition: Path queries
- Centroid Decomposition: Tree divide and conquer
