## Key Concepts

### Graph Traversals
- DFS: Tree edges, back edges, forward edges, cross edges
- BFS: Level-by-level, multi-source BFS, 0-1 BFS
- Topological sort: Kahn's algorithm, DFS-based

### Shortest Paths
- Dijkstra: Non-negative weights
- Bellman-Ford: Handles negative weights
- Floyd-Warshall: All-pairs shortest paths
- A*: Heuristic-guided search

### Advanced Topics
- Network Flow: Ford-Fulkerson, Edmonds-Karp, Dinic's
- 2-SAT: Implication graph + SCC
- Euler paths: Hierholzer's algorithm
